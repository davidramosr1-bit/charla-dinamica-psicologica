# La salud mental en la Fe — Charla Psicológica v2

**Hna. Andrea Moys de Alfaro & Hna. Keren de Alvarenga**

## 9 Diapositivas

1. **Portada** — "La salud mental en la Fe"
2. **Dinámica corporal** — Alegría, Ansiedad, Tristeza, Confianza con gestos
3. **Video** — Pausa para reproducir video externo
4. **Quiz interactivo** — 4 escenarios + QR para participantes
5. **Imagen** — ¿Qué no es la ansiedad?
6. **Imagen** — ¿Qué es la ansiedad?
7. **Lluvia de ideas** — Expositora escribe ejemplo (zoom interactivo)
8. **QR participantes** — Envían sus palabras desde el celular
9. **Universo de respuestas** — Palabras aparecen en vivo

## Participantes (desde celular)
- Menú con 2 actividades: Quiz y Lluvia de ideas
- No requiere cuenta ni app
- Las respuestas llegan en tiempo real

## Despliegue

1. Subir a GitHub
2. Importar en Vercel
3. Conectar Upstash Redis (Storage → Upstash → Redis)
4. Variables necesarias: `UPSTASH_REDIS_REST_KV_REST_API_URL` y `UPSTASH_REDIS_REST_KV_REST_API_TOKEN`
5. Redeploy

Contraseña expositora: `salud2026` (editar en `app/data.js`)
